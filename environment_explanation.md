## The environment explained (a bit)

You can create an environment on your machineby running our instructions in the readme. We are testing to make sure our environment file is up to date sporadically- and that it is all needed to run all the notebooks. 

Some dependancies e.g. pandas should be dragged in by other dependancies. However if you are creating an environment by hand, you may want to check that your environment matches ours (more or less). Also, as the codebase evolve occasionally the environments neccesary to run our work can diverge before we re-align them. Therefore we here include a Window's environment dump print-out from two the builders of the library. Please note these may not match the shared environment we test on, but should also run the individual's work:
